# Digital Water Footprint Calculator

A full-stack project built using Node.js, Express, MongoDB, and Vanilla JS.

## Features
- Food category icons
- Water footprint calculator for food items
- Search functionality
- Feedback + Request Forms (MongoDB)
- Local image rendering

## Tech Stack
- Node.js + Express
- MongoDB
- HTML/CSS/JS (Vanilla)
- GitHub for version control

## Setup Instructions
1. Clone the repo
2. Run `npm install` in backend
3. Start MongoDB locally
4. Run server using `node server.js`
5. Open `frontend/index.html`
